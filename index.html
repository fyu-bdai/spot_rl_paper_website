High-Performance Reinforcement Learning on SpotMini: Optimizing Simulation Parameters with Distributional Measures

AJ Miller1,2, Fangzhou Yu1, Michael Brauckmann1, Farbod Farshidian1
RAI1, MIT2
ICRA 2025
Paper Code arXiv

Abstract

This work presents an overview of the technical details behind a high-performance reinforcement learning policy deployment with the Spot RL Researcher Development Kit for low-level motor access on Boston Dynamic’s SpotMini. This represents the first public demonstration of an end-to-end reinforcement learning policy deployed on Spot hardware with training code publicly available through Nvidia IsaacLab and deployment code available through Boston Dynamics. We utilize Wasserstein Distance and Maximum Mean Discrepancy to quantify the distributional dissimilarity of data collected on hardware and in simulation to measure our sim-to-real gap. We use these measures as a scoring function for the Covariance Matrix Adaptation Evolution Strategy to optimize simulated parameters that are unknown or difficult to measure from Spot. Our procedure for modeling and training produces high-quality reinforcement learning policies capable of multiple gaits, including a flight phase. We deploy policies capable of over 5.2m/s locomotion, more than triple Spot’s default controller maximum speed, robustness to slippery surfaces, disturbance rejection, and overall agility previously unseen on Spot. We detail our method and release our code to support future work on Spot with the low-level API.

Overview

Visual overview of the training and deployment pipeline. On the left, is an approximation of how the policy is trained in simulation, and on the right, is how it’s deployed on hardware. IsaacLab is the simulation environment, provides the physics engine, and employs our modeling for training. We deploy our policies on Spot using an Nvidia Jetson Orin that communicates with the Boston Dynamics API for state estimates and sends motor control law values with the Spot RL Researcher Development Kit to command the motors.
